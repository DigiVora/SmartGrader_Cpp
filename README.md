## 🎓 SmartGrader-Cpp

**SmartGrader-Cpp** adalah aplikasi berbasis Command Line Interface (CLI) yang dirancang untuk memudahkan pengelolaan dan rekapitulasi nilai mahasiswa. Menggunakan konsep **Array of Struct**, aplikasi ini memastikan pengolahan data dilakukan secara terstruktur dan efisien.

## 🚀 Fitur Utama
*   **Input Dinamis**: Menentukan jumlah data mahasiswa sesuai kebutuhan di awal program.
*   **Automated Calculation**: Menghitung Nilai Akhir secara otomatis dengan bobot: (40% UAS + 60% UTS).
*   **Grade Determination**: Penentuan nilai huruf otomatis (A - E) berdasarkan standar akademik.
*   **Responsive Table**: Tampilan tabel yang rapi dan presisi menggunakan library `<iomanip>`.
*   **Auto-Clear Screen**: Tampilan bersih dengan fitur `system("cls")` sebelum hasil ditampilkan.

---

## 🛠️ Teknologi yang Digunakan
*   **Bahasa Pemrograman**: C++
*   **Konsep**: Struct, Array, Function, I/O Stream, Manipulator.
*   **Library**: `<iostream>`, `<string>`, `<iomanip>`.

## 💻 Cara Menjalankan
1. Clone repository ini:
   ```bash
   git clone [https://github.com/DigiVora/SmartGrader-Cpp.git](https://github.com/DigiVora/SmartGrader-Cpp.git)
   ```
2. Buka file main.cpp di IDE Favorit Anda (Dev-C++, NetBeans, atau VS Code).
3. Compile dan Run Program.
4. Masukkan jumlah mahasiswa dan data nilai yang diminta.
